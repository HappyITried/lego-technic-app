# LEGO Revuelto Controller

Custom app to control the LEGO Technic 42214 Lamborghini Revuelto
with a PS4 / Xbox / generic controller or keyboard.

## Download
- Download the `.exe` (Windows) — no Python required
- Or run the Python script directly (see below)

## How to use

1. **Pair your LEGO hub with your PC first** via Windows Bluetooth settings before opening the app.
2. Power on the Revuelto and press the green button until it blinks.
3. Launch the app — it will scan and connect automatically.
4. Once connected, use your controller or keyboard to drive.

> Make sure Bluetooth is enabled on your PC.

### Keyboard controls (default)

| Action | Key |
|---|---|
| Forward | W |
| Reverse | S |
| Steer Left | A |
| Steer Right | D |
| Brake | Space |
| Cycle Lights | L |
| Cycle Speed Mode | Left Shift / Left Ctrl |
| Toggle Fullscreen | F11 |

All binds are remappable in the built-in Keybind Editor.

### Speed modes

| Mode | Description |
|---|---|
| NORMAL | Balanced everyday driving |
| SPORT | Full throttle |
| SLOW | Careful driving |
| GRANDMA | Very slow |
